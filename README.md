# La Maison des Pyjamas — Site officiel

Site complet pour **La Maison des Pyjamas**, boutique basée à Cotonou.

> Ce projet n’est pas pensé comme un e-commerce classique avec paiement en ligne dès le départ. Il s’agit d’un site complet de marque : présentation, catalogue éditorial, collections, informations pratiques, contact, commande via WhatsApp et espace de mise à jour simple des produits.

## Objectif

Créer une présence web premium, chaleureuse et festive pour La Maison des Pyjamas, avec une expérience spéciale Noël : pyjamas, famille, confort, cadeaux, ambiance cosy et commandes rapides.

## Positionnement

La Maison des Pyjamas vend plus qu’un vêtement : elle vend le confort, la douceur, les souvenirs de famille et l’ambiance des fêtes.

## Pages prévues

- Accueil
- Collections
- Catalogue
- Noël à la Maison
- À propos
- Guide tailles
- Livraison & commandes
- Contact
- FAQ
- Administration produits

## Fonctionnalités prévues

- Catalogue produits administrable
- Mise à jour des produits vendables
- Catégories : Femme, Homme, Enfant, Famille, Noël
- Produits mis en avant
- Badges : Nouveau, Noël, Best-seller, Stock limité
- Commande via WhatsApp
- Galerie Instagram / réseaux sociaux
- Design mobile-first
- Ambiance Noël premium

## Stack recommandée

- Next.js
- TypeScript
- Tailwind CSS
- Supabase pour les produits
- Cloudinary ou Supabase Storage pour les images
- Déploiement Vercel ou Netlify

## Direction artistique

Palette issue du logo et de l’univers Noël :

- Rose logo : `#E91786`
- Noir profond : `#080808`
- Blanc chaud : `#FFF8F0`
- Rouge Noël : `#B11226`
- Vert sapin : `#0F5132`
- Doré doux : `#D8B86A`

## Réseaux à intégrer

Les liens officiels seront ajoutés dans `content/brand.json` :

- Instagram
- TikTok
- Facebook
- WhatsApp
- Google Maps

## Structure du contenu

Les informations de marque sont centralisées dans `content/brand.json`.
Les produits de départ sont dans `content/products.json` en attendant la connexion Supabase.
